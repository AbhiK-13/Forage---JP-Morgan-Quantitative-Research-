# Forage---JP-Morgan-Quantitative-Research-

This repository contains my attempts at the JP Morgan course on Quantitative Research on Forage. 

The tasks involved creating forecasts and pricing functions using time series data on natural gas prices, as well as analysing a book of loans to estimate a customer's probability of default, and using dynamic programming to convert FICO scores into categorical data to preidct defaults. 

Doing this program confirmed that I really enjoy manipulating large data sets to inform business solutions, and built my data analysis and modelling skills in a real-world context. Additionally, it helped provide a deeper insight into the volatility/behaviour of commodities.
